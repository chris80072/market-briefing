# 台股盤前簡報 (Taiwan Stock Pre-Market Briefing)

每日在美股收盤後，為台灣投資人彙整國際新聞與市場資訊，提供台股投資決策參考。

## 📋 報告內容

每份台股盤前簡報包含以下欄位：

### 1. 美股與國際市場收盤速覽
- 美股三大指數（道瓊、標普 500、那斯達克）
- 費城半導體指數（SOX）
- 重要 ADR（台積電、聯電、日月光等）
- 10Y 美債殖利率、美元指數、VIX 恐慌指數
- 油價、黃金、銅等大宗商品
- 新台幣兌美元匯率

### 2. 五大重點與影響分析
精選最可能對台股產生重大影響的五則資訊，每則包含：
- **出處** — 原始新聞來源連結
- **事件摘要** — 發生了什麼事（附具體數字與來源）
- **對台股影響** — 影響哪些類股與個股，方向為何
- **風險提醒** — 後續需留意的風險因素
- **投資建議** — 操作方向與策略

### 3. 台股關注重點
- **權值股焦點** — TSMC、鴻海等主要持股
- **類股輪動觀察** — 強勢/承壓/機會類股
- **技術面與籌碼面** — 關鍵支撐、外資進出、融資餘額
- **重要行事曆** — 當日及近期重要公告、財報、經濟數據

### 4. 綜合結論與操作建議
- **開盤預判** — 偏多/偏空/震盪，附說明理由
- **策略建議** — 按投資風格分類（保守/波段/長期）
- **需警戒的風險事件** — 未來 24-72 小時關注事項

### 5. 資料查證備註
- 所有數字來源標示
- 未能查證的項目標記「尚待確認」

## 🎯 使用方式

1. **按日期查找** — 每份報告以 `YYYY-MM-DD.md` 命名
2. **按月份瀏覽** — 進入 `年份/月份/` 資料夾查看該月所有報告
3. **快速查詢** — 使用 GitHub 搜尋功能找特定主題或個股

## ⏰ 更新時間

- 每個台股交易日產生一份報告
- 通常在美股收盤後（台灣時間次日凌晨）生成
- 若台股休市，報告內容會以「下次開盤日」為視角撰寫

- 所有數據盡力查證，但不保證絕對準確
