# 市場簡報 (Market Briefing)

每日彙整台股與美股的重點資訊、技術面分析與投資建議。

## 📂 資料夾結構

- **`tw_stocks/`** — 台股盤前簡報
  - 每日彙整美股收盤後的國際新聞與市場資訊
  - 列出最可能對台股產生重大影響的五則重點
  - 包含技術面、籌碼面分析與操作建議

- **`us_stocks/`** — 美股市場簡報
  - 美股及全球市場行情速覽
  - 重大財經新聞與地緣政治事件分析
  - 市場展望與投資策略建議

## 📅 更新頻率

- **每個交易日產生新報告**
- **按年份 → 月份 → 日期組織** (`YYYY/MM/YYYY-MM-DD.md`)

## 🗂️ 使用方式

```
market-briefing/
```

## ⚠️ 免責聲明

本報告由 AI 彙整公開資訊與新聞，**僅供參考**，不構成投資建議。

- 所有數據與資訊來自公開來源，已盡力查證但不保證絕對準確
- 投資涉及風險，投資人應自行評估並承擔投資決策的責任
- 過往表現不代表未來績效

## 📝 內容說明

每份報告包含以下主要欄位：

1. **市場收盤速覽** — 主要指數、匯率、商品價格等
2. **五大重點分析** — 最具影響力的市場事件
3. **台股/美股關注重點** — 類股輪動、技術面、行事曆
4. **綜合結論與操作建議** — 開盤預判與策略建議
5. **資料查證備註** — 數據來源與查證狀況
