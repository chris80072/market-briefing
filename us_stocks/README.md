# 美股市場簡報 (US Stock Market Briefing)

每日彙整美股及全球市場的重點資訊、市場動態與投資分析，為投資人提供美股投資決策參考。

## 📋 報告內容

每份美股市場簡報包含以下欄位：

### 1. 美股與全球市場收盤速覽
- 美股三大指數（道瓊、標普 500、那斯達克）收盤點位與漲跌
- 道瓊運輸指數、羅素 2000 小型股指數
- 科技股（Nasdaq-100）、標普 500 成長股表現
- 國際股市（日股、英股、歐股等）
- 美國 10Y 公債殖利率、高收益債券利差
- 美元指數、主要貨幣對
- VIX 恐慌指數、商品期貨（油、金、銅）
- 加密貨幣（比特幣、以太幣）

### 2. 五大重點與市場分析
精選最具影響力的五則資訊，每則包含：
- **出處** — 原始新聞來源連結
- **事件摘要** — 發生了什麼事及其市場背景
- **市場影響** — 對不同類股、主要個股的衝擊
- **風險評估** — 需關注的後續發展與風險因素
- **投資建議** — 策略調整方向與風險控管建議

### 3. 美股類股與個股觀察
- **強勢類股** — 表現領先的行業
- **受壓類股** — 面臨調整的行業
- **重點個股** — 大型科技股、金融股、工業股等動向
- **技術面** — 主要指數的關鍵支撐與壓力位
- **基本面** — 重要財報、經濟數據、企業動向

### 4. 聯準會與央行動態
- 利率決議與政策展望
- 主要央行官員言論
- 市場對未來政策的預期
- 通膨與就業數據解讀

### 5. 地緣政治與宏觀經濟
- 全球貿易與地緣政治事件
- 能源與原物料價格波動
- 新興市場與匯率動向
- 經濟數據與市場預期差異

### 6. 綜合結論與操作建議
- **市場展望** — 短期與中期前景
- **風險評估** — 主要下行風險
- **投資策略** — 資產配置與個股建議
- **待觀察事項** — 未來關鍵時間點與事件

## 🎯 使用方式

1. **按日期查找** — 每份報告以 `YYYY-MM-DD.md` 命名
2. **按月份瀏覽** — 進入 `年份/月份/` 資料夾查看該月所有報告
3. **追蹤特定主題** — 使用 GitHub 搜尋功能找特定央行、地緣政治事件或個股

## ⏰ 更新時間

- 每個美股交易日產生一份報告
- 通常在美股收盤後（東部時間下午 4 時）生成
- 若美股休市，報告內容會說明休市原因及下次開盤時間

## 💡 報告特色

- ✅ **全面性** — 覆蓋指數、商品、匯率、央行、地緣政治等多維度
- ✅ **實時性** — 基於當日最新市場數據與新聞
- ✅ **可追溯性** — 每個數字都附上來源連結
- ✅ **可操作性** — 提供具體的投資策略建議與風險提醒
